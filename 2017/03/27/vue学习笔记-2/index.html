vuejs组件之间的调用

假设我们有一个入口和两个组件

  • App.vue — 项目入口
  • footer.vue — footer组件
  • header.vue — header组件

其他组件要想被app.vue调用需要在script标签里面写export default

而在App调用其他组件的方法要使用import和components

先用import引入组件

然后用components注册

最后就可以在app.vue的template里插入
